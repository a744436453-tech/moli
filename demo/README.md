# Moli 前端 Demo

## 预览方式

直接打开：

```text
demo/index.html
```

无需安装依赖。

## 当前包含的功能板块

1. 首页陪伴
2. 生成数字形象
3. Momo 聊天
4. 回忆时间线
5. 互动玩法
6. 我的与设置

## 交互说明

1. 左侧导航可切换不同功能板块。
2. 页面中的主要按钮已添加即时反馈。
3. 手机框中的页面用于表达移动端核心结构。
4. 后续可将 `原型图/` 中的真实视觉稿替换到对应板块。

## 后续可继续扩展

```text
demo/
├── index.html        # 当前整合式 Demo
├── styles.css        # 视觉样式
├── app.js            # 交互逻辑
└── pages/            # 后续可拆二级页面
```
